1. data quality assessment

* mismatched data types
* mixed data values
* data outliers

2. data cleaning

* missing data

	-> manually fill in missing data

* noisy data

	-> regression
	-> clustering

	-> removing HTML tags, boilerplate email text, unnecessary blank texts btw words
	-> removing dublicate data



3. data transformation

	-> aggregation
	-> normalization
	-> feaature selection
	-> discreditization
	-> conecpt hierarchy generation



4. data reduction

	-> attribute selection
	-> numerosity reduction
	-> dimensionality reduction
